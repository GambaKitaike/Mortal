## 結論

- **崩落区間 step6000→8000**: 副露和了率 38.24% → 20.58% (-17.65pp)、立直和了率 61.50% → 78.99% (+17.49pp)。
- B1: P(call|legal) 30.42% → 15.37% (-15.06pp)、P(chi|legal) -4.66pp、P(pon|legal) -27.13pp。
- B2: P(和了|副露局) 17.16% → 15.96% (-1.20pp)、流局 +5.10pp、他家和了 -3.90pp (放銃 -0.91pp、横移動 -2.99pp)。
- 副露局平均鳴き回数: 1.938 → 1.818 (-0.119)。

### 決定木判定
- **主因: 行動抑制** — P(call|legal) が 6000→8000 で大きく低下。 CQL/報酬が鳴き行動を直接潰している。 対処: CQL 強度の step 依存 or 鳴き行動への保護。

### sanity

- B2 母数: 各 step で 副露局+門前局 == trainee が打った局数（total_kyoku）。
- 排他: 副露局結末 trainee和了+他家和了+流局 == 副露局数。
- 放銃+横移動 == 他家和了（副露局）。

### action ID

- 副露: trainee action ∈ {38, 39, 40, 41, 42}
- 立直: trainee action == 37
