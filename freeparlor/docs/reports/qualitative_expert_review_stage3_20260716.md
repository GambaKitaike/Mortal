## 所見（Gamba、1 半荘時点）

1. **鳴き頻度の増加と新レパートリー**: 鳴きは確かに増えていた。Stage1 では見られなかった
   **喰いタン仕掛け**を実行。**役バックの仕掛け**も出現（評価点）— ただしその後、
   **当の役牌を自分から切って和了役を消していた**（何の役で和がるつもりなのか不明な
   打牌）
2. **鳴いた後の続行は下手なまま（Stage1 から不変）**: ポンテンを取らずスルーする場面を
   複数回目視。また鳴き手でシャンテン数を不必要に下げる打牌も複数回 — 鳴き手は面前と
   違って良型に受け直すインセンティブが薄いのに、面前手のような手変わり志向を続ける
3. **カン周りの判断が顕著に異常**: 役牌暗刻でテンパイしている局面で、4枚目の役牌が
   切られた際に**ポンしてわざわざテンパイを崩した**（暗刻から2枚を晒すポン。カンでも
   スルーでもなく最悪の選択肢）。また自分が良型立直中に、暗槓できる4枚目を
   **暗槓せずツモ切り**（裏・新ドラの期待を放棄、これも損に見える）。
   カン局面は出現頻度が低く学習しきれていない可能性（レビュアー仮説）
4. **降りの規律が崩壊気味**: もはや「降りている」と言えない打牌が多い。途中まで現物を
   切っていたのに突然危険牌を切って放銃するパターンを頻繁に目視。Stage1 比での
   放銃率上昇と整合する牌譜像

## 監督側の注記

- (1)(2) は「**鳴きの入口は増えたが、鳴いた後のサブゲームが未熟**」という構図。
  鳴き後の局面は訓練分布上稀（Stage1 平衡はほぼ鳴かず、Stage3 も anneal 後は副露率
  14.66% まで回帰）なので、call_bonus が入口の頻度だけ押し上げても鳴き後状態の
  学習量が不足する、という state-coverage 仮説が立つ（未検証）
- 同時に **DRCA セット(b) の解釈への注意材料**: セット(b) は「鳴きスキルの高い
  checkpoint ほど同一分岐点から価値を引き出せるか」を測るが、本所見は Stage3-16000 の
  鳴きスキルが**実行品質では高くない**（入口だけ増えた）可能性を示す。セット(b) で
  Stage3-16000 が正側に出ない場合、この続行力不足が説明候補になる
- (3) のカン局面のサンプル希少性は計測側でも既知（`verify_ppo_p1.py` 検定(15) の
  daiminkan flake、DRCA §1 amendment の own-turn kan ケース）。大明槓の報酬配置
  **配管**は検定(15)で検証済みなので、これは配管バグではなく方策の未学習という
  切り分けができている
- (4) は Stage1 所見(2)（押し引きの歪み: ベタオリ時の非現物選択）から**悪化**した
  可能性がある。メタ対決の結果（Stage3-16000 vs Stage1-16000 でチップ −2.9SE、
  残留鳴きが立直メタで負債）と整合する牌譜像
- **0b 議題1（商品性）への含意**: 「鳴きが増えたが下手」は商品性の観点で最悪の象限 —
  役消し・テンパイ崩しポン・降り放棄は、いずれも人間の目に**明確な失着として映る**
  クラス。argmax 配備候補としての見栄えは Stage1-16000 の方が失着が目立ちにくい
  方向の所見（1 半荘時点の暫定）
