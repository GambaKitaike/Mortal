## 3. 結論

> **Q1: 宣言行動の平均 advantage は非宣言より系統的に低いか？（raw / 正規化後それぞれ）**

**弱い Yes（raw のみ・鳴き）。立直は同程度。正規化後は系統差なし。**

400 step 全期間を step 重み付き集計:

| カテゴリ | raw mean | raw std (pooled) | n steps |
|---|---:|---:|---:|
| 鳴き実行 | 0.276 | 6.85 | 4,794 |
| 鳴き可能・不実行 | 0.294 | 7.40 | 15,137 |
| 立直実行 | 0.421 | 8.23 | 291 |
| 立直可能・不実行 | 0.513 | 8.09 | 550 |

- **raw:** 鳴き実行は不実行より **−0.018**（方向はペナルティ仮説と一致するが std≈7 に対し無視できる差）。
  立直も実行 < 不実行（−0.092）だが n=291 vs 550 とサンプルが小さい。
- **正規化後 (per-batch):** 鳴き −0.022 vs −0.037、立直 −0.022 vs −0.029 — いずれも |Δ| < 0.02。
  per-batch 正規化のため batch 間集計での解釈は限定的。
- **時系列:** early(0–99) では鳴き raw 0.644 vs 0.743、late(300–399) では 0.014 vs 0.197 と
  **両群とも低下**（critic 未収束下の全体的 GAE スケール縮小）。実行群の相対差は late でやや拡大するが、
  400 step 全体では「系統的に低い」とは言い切れない。

> **Q2: 低い場合、報酬3ストリームのどれが主成分か？**

**鳴き局では3ストリームすべて低下。素点Δと GRPΔ が同程度、チップも追随。**

局単位（4136 局、step 重み付き）:

| 条件 | n | 素点Δ/1000 | GRPΔ | chip×5 |
|---|---:|---:|---:|---:|
| 鳴きあり | 2,871 | 0.566 | 1.378 | 0.693 |
| 鳴きなし | 1,265 | 1.143 | 2.185 | 1.336 |
| **差 (鳴き−非鳴き)** | | **−0.577** | **−0.807** | **−0.643** |

- 鳴き局は非鳴き局より **素点Δ −0.58 / GRPΔ −0.81 / chip −0.64**。
- 総合 blend (α=γ=β=1) でも約 **−2.03 / 局**。主成分は **GRPΔ やや大** だが、
  素点Δも同符号・同程度 — **H1（素点レジーム）単独説明でも GRP-OOD 単独説明でもない**。
  設計書 §6 の GRP 懸念は「鳴き選択局」では GRP が最大項だが、素点も連動して下がる。

立直局は advantage 上はやや低いが、**実現報酬は立直局の方が高い**（選択バイアス）:

| 条件 | n | 素点Δ/1000 | GRPΔ | chip×5 |
|---|---:|---:|---:|---:|
| 立直あり | 281 | 1.001 | 2.153 | 1.139 |
| 立直なし | 3,855 | 0.723 | 1.586 | 0.872 |

→ Q2 の「advantage が低い → どのストリームか」は **鳴きに限れば答えられる**。
立直は advantage と realized reward の符号が逆（良形で立直する confound）。

> **Q3: 立直局の素点Δ平均は非立直局よりどれだけ低いか（供託・放銃込みの実測）**

**低くない。立直局の方が +0.278 / 1000 点（≈ +278 実素点）高い。**

- 立直局 mean 素点Δ/1000 = **1.001**
- 非立直局 = **0.723**
- 差 = **+0.278**（供託・放銃ペナルティを期待する方向と逆）

解釈: 400 step・自己対戦では「弱い形で立直して供託/放銃」より「形が良いときに立直」が支配的。
供託・放銃込みの net ペナルティはこの集計では **出ていない**（少なくとも平均では相殺以上）。

## 4. 総評

- P2b 同 config + 計装のみで 400 step 完走。`advantage_decomp` / `kyoku_reward_decomp` / `action_mass(n)` は期待どおり 400 行/step。
- **鳴き:** raw advantage は実行 < 不実行だが差は小さい。実現報酬は3ストリームすべて鳴き局で低く、
  GRPΔ がやや最大だが素点Δも連動 — 単一ストリーム支配ではない。
- **立直:** advantage 差は弱い。素点Δは立直局の方が高く、Q3 の供託/放銃ペナルティ仮説は本 run では支持されない。
- 次段: advantage 差が late step で拡大する傾向あり → 長 run または critic 安定後の再計測が有効。
