**位置づけ: 定性・exploratory・判定非関与。** DRCA 本測定の判定は
`drca_probe_design.md` §5a の凍結条件（全体平均 + 事前列挙 contrast）のみで行い、
本書はそれに一切関与しない。frame1（セット(a)×Stage1-16000、N=485、
ΔQ̄=−3.2635±0.7208）の分岐点別 ΔQ̂ の正負極値 各12件を、専門家（Gamba、麻雀上級者）が
牌譜 HTML で全件目視評価した記録。**測定器の外的妥当性チェック**（ΔQ̂ の極値が
専門家の善悪判断とどの程度一致するか）として、Stage1/Stage3 argmax レビュー
（`qualitative_expert_review_20260715.md` / `qualitative_expert_review_stage3_20260716.md`）
の系譜に連なる資料。

## 0. 方法

- 分岐点別 ΔQ̂ = mean(call 腕 K=8) − mean(no_call 腕 K=8)。全体平均が commit 済み
  aggregate（−3.2635）と一致することを assert した上で |ΔQ̂| 上位を抽出
- 局表示（何局何本場）は牌譜 start_kyoku イベントから機械解決。巡目 = `at_turn`
  （局開始からの自席ツモ回数、libriichi `update.rs` で tsumo 時のみ +1）。
  鳴き対象牌は手牌追跡による近似特定（own_tsumo 一致 + 鳴き種一致）
- 詳細テーブル: `runs/drca/main_s1final_20260715_075837/frame1_extremes_summary.md`
  （ローカル成果物）。牌譜 HTML: `runs/viewer_out/`
- 注意: 個別分岐点の ΔQ̂ は腕内 SE が大きく anecdotal。1局に複数の鳴き機会があっても
  測定分岐点は1局1点の一様乱択で選ばれた1つだけ（レビュー時に別機会と取り違えた
  事例が実際に発生 → 対象牌列の追加で解消）

## 3. 監督側の総合

1. **非対称性**: 負の裾は純度が高く（~9/12 が本物）、正の裾は半分が結果論。
   ΔQ̄ < 0 の主シグナル（機会費用 = 手壊しコスト）は専門家の目でも頑健。
   「取りこぼし」側のシグナルは実在する（本物4件）が希釈されている
2. **「結果論」の正体は方策条件付きの真実である可能性**: 正側の結果論5件はほぼ全て
   「パスした未来で方策がリーチに突っ込んで放銃」型。ΔQ̂ は当該方策の続行力での
   反実仮想なので、方策の降りが雑（Stage1 所見(2)・Stage3 所見(4)で確認済み）である
   限り、「鳴いて局を畳む方が*この方策には*得」は測定として正しい。
   人間基準の善悪との乖離は測定バグではなく**外的妥当性の限界**
   （`qualitative_expert_review_20260715.md` 監督注記の再確認）
3. **負側 #3 は競技力ギャップ仮説（解釈C）のメカニズムの目視実例候補**:
   専門家は「めちゃくちゃ鳴きたい」局面で、負値はポン後の危険牌押し（5m）に由来。
   つまり**鳴き自体ではなく鳴いた後の続行の下手さが鳴きEVを押し下げている**構図。
   セット(b)（trainee=Stage3-16000 で同じ分岐点を再測定）が動かそうとしている
   変数がまさにこれ — セット(b) の結果を読む際の具体的な参照点になる
4. 測定手順上の教訓: 分岐点の提示には**対象牌（誰の何切りか）が必須**
   （巡目と局だけでは同一局内の別機会と取り違える）。`at_turn` の定義
   （自席ツモ回数）と表示巡目の ±1 ずれも2件で指摘あり — 以後の枠の
   極値レビューでも同形式（対象牌付き機械生成テーブル）を用いる
