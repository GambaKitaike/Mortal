個別ノート（HTML と `drca_pilot_qualitative_drilldown.py` の出力で追跡可能）:

- **負の極値（0・44・2）は共通して「no_call 側の未来に自分の和了がある」局面**。
  特に branch 44 はオーラス2着目で、鳴かず進めた元の対局はロン和了で逆転トップ。
  序盤の何気ないチーがこの未来を壊す — 機会費用ギャップ（~5.5–5.9 チップ）の実体が
  1 局面に凝縮された例
- **branch 30 は「トップ目の鳴きはリスクだけ買う」型**。元の局は他家間の横移動で
  自分は無傷（0）。call 腕平均 −25.4: 鳴いて前に出ると失点側に回る
- **正の極値は2類型**: (a) **branch 24 = 役牌+赤2枚の明白な好機** — これは方策が
  実際に鳴いており（orig_action=41）、親ツモ +12700 に繋がった。方策は「教科書的な
  好機」を既に取れている。(b) **branch 15 = 親ラッシュへの防御的速度鳴き** — 両腕とも
  負けだが、鳴くと損失がほぼ半減（−16.1 vs −34.6）し、d_chip は +16.25 と全項目中最大。
  「鳴いて局を畳む」価値はチップストリームに強く乗る。この類型は方策がパスしており、
  **取りこぼしがあるとすればこの象限**という作業仮説が立つ（exploratory）
- **branch 12（ΔQ̂ 恒等 0）**: 巡目14・赤3枚保持でも、局の結末が既に確定していて
  分岐が無効力なケースが存在する。この種のデッドポジション混入は全体 ΔQ̄ を 0 側へ
  薄める（設計上は N と cluster SE が吸収する話で、判定への含意はない）

**総観（exploratory）**: パイロット平均が n.s.（−0.46±1.59）なのは「手を壊す大負」と
「速度・役確定の大勝」がほぼ拮抗している姿と整合的。本測定 N=485 は
この拮抗の勝者を CI 半幅 ≤1.0 千点で分解する。0b セッションでは、仮に全体 ΔQ̄ ≈ 0
（シナリオ N）でも branch 15 型（防御的速度鳴き）の層が正に残るか、が
exploratory な注目点になる。
