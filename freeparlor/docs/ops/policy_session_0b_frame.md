## 0. セッションで決めること（3+1 議題）

1. **立直マキシマリズムの商用採否** — 現行均衡（鳴き最小・立直特化）の方策を
   フリー雀荘 AI として採用するか
2. **経済定数変更** — ウマオカ・チップ単価・β 等を動かす新実験系を開くか
3. **敵対的搾取者訓練の要否** — 現行均衡を搾取する方策を訓練して頑健性を検査するか
4. （付随）**Stage2b 解凍実験の再評価**（バックログ3） — 配備税の発見で価値が
   上がった「収束済み方策の分布シフト適応」

## 1. 確定済みの事実ベース（DRCA 結果に依存しない）

- **探索ラダー3段全て不成立**: Stage1 純探索（0.236×）→ Stage2 分布介入（0.185× +
  配備税）→ Stage3 報酬介入（anneal 後 1.066×・有意減衰）。立直マキシマリズムは
  経路によらない頑健な均衡
- **正典反鳴き勾配は定量再現**: 判定窓 raw advantage は call_taken +0.018 vs
  declined +0.522（Stage3）≈ Stage1（+0.012 / +0.524）
- **機会費用ギャップ ~5.5–5.9 チップが支配項**。副次確認: 鳴き局正典収支は改善方向
  （チップ/局 −0.95 → −0.63 → −0.33）だが損益分岐に届かず
- **配備税は介入形態依存**: 分布介入（Stage2）は有意、anneal 内蔵の報酬介入
  （Stage3）はゼロ → 「介入は anneal とセット」制約は対照実験済み
- **メタ対決**: 残留鳴きは立直メタで負債（Stage3-16000 vs Stage1-16000 チップ
  −0.675±0.236 ≈ −2.9SE）
- **DRCA パイロット**: ΔQ̂ = −0.4575 ± 1.5882 千点（n=50、n.s.）。本測定 N=485 で
  95%CI 半幅 ≤1.0 千点まで絞られる
- **§5a-1b 適用済み**: セット(b)×init 脱落。解釈 C は
  「Stage3-16000 vs Stage1-16000（セット(b)）」の1本で判定

## 2. DRCA 判定 × 議題の分岐（討議用シナリオ）

主判定はセット(a)×主レンズの ΔQ̄（4 checkpoint）+ contrast 2/3（§5a-2）。
解釈 A–D の条件文言は `drca_probe_design.md` §4（凍結）が正。

### シナリオ A — 内在的機会費用（全 checkpoint で ΔQ̄ < 0 有意、(b) で正側移動なし）

- 技術的含意: 方策は正しく損を学んでいる。phase 2（DRCA 訓練組み込み）は
  正当化されない（§4 A の凍結文言どおり）
- 議題1: 立直マキシマリズムは**当該経済設定での正解**。採否は「鳴かない瑕疵」では
  なく絶対競技力・商品性で判断する土俵に移る
- 議題2: 「鳴きのある AI を見たい」なら**経済定数変更が唯一の残存経路**
  （感度実験系の設計へ）
- 議題3: 独立に価値あり（均衡の頑健性検査として）

### シナリオ B — credit assignment 失敗（ΔQ̄ > 0 有意 かつ π(鳴き|可能∧赤保持) < 0.31）

- 技術的含意: 学習器が「得」を見えていない。phase 2 の検討解禁（§7）
- 議題1: 商用化は保留寄り（伸び代が残っている）。phase 2 の成否を先に見る選択肢
- 議題2: 定数変更の前に学習器側の修理が先
- 議題3: 価値上昇（搾取可能性が実在する示唆）

### シナリオ C — 競技力ギャップ（(b) で Stage3-16000 が Stage1-16000 より有意に正側）

- 技術的含意: 鳴きスキルを付ければ得になる。**C∧B は最高介入価値の象限**（§4）
- 議題1/2/3: B に準じるが、介入手段はスキル訓練（カリキュラム等）が第一候補になる

### シナリオ D — レンズ間符号割れ（主・副レンズが両方有意で逆符号）

- 技術的含意: 報酬設計誤指定の疑い → `reward_design_teacherfree.md` 再訪を起票
  （§4 D）。**議題1–3 は全て凍結し再監査を優先**

### シナリオ N — 全て n.s.（0 近傍・検出力は充足）

- 技術的含意: 赤保持鳴きの機会費用 ≈ 0（鳴いても鳴かなくても大差ない）
- 議題1: 「鳴かない損」も「鳴く得」もない → 純粋に商品性（見た目の打牌选択）の
  問題に還元。ε-greedy 的な演出混合等の非学習的解も土俵に入る
- 議題2/3: 動機は弱まる
