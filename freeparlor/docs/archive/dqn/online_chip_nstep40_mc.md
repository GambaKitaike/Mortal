## 所見

- bootstrap 除去（純 within-kyoku MC）でも step8000 以降の 副露Δ・鳴き和了率の時系列振動は残存。drift の主因は bootstrap 非定常だけでは説明できない。
- n=40 は step16000 で 副露Δ=−0.03pp と mqw03（−2.47pp）より 0 付近に収束。振動の「質」は異なり、局末打ち切り MC では late-stage の副露抑制方向への収束が見える。
- 鳴き和了率は両 run とも step6000 以降 2–3% 台に落ち込み、MC 化でも回復せず。chip ヘッドの drift とは独立した policy/CQL 側の問題が疑われる。
- 全体副露率は n=40 の方が低く振幅も小さい（22–31% vs 27–43%）。放銃率・avg_rank は両 run とも安定で、MC 化による安全性悪化はなし。
- chip実現率は両 run とも 19–23% 帯で同等。bootstrap target の有無は chip 実現そのものより、副露・鳴き和了の時系列安定性への影響が限定的。
